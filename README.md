# 🚀 The Wild Oasis

A modern web application built with a powerful frontend stack, focused
on performance, scalability, and great developer experience. <br>

## 🧰 Tech Stack

This project is built using:

React – UI library for building interactive interfaces <br>
TypeScript – Static typing for safer and more maintainable code <br>
React Query – Data fetching, caching, and server state management <br>
Supabase – Backend-as-a-Service (database, auth, APIs) <br>
Styled Components – CSS-in-JS for component-based styling <br>
React Hook Form + Zod – Forms handling with schema validation <br>
React Router – Client-side routing <br>
